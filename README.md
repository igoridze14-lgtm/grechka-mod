# Grechka Companion Mod 🐕

## Что это?
Мод для Minecraft Fabric 1.21.1 который добавляет в игру **бессмертную собаку-компаньона** по имени **Grechka**!

## Особенности
✅ **Бессмертная собака** - не может умереть от урона
✅ **Приручение** - используй **костью** чтобы приручить собаку
✅ **Следует за игроком** - собака будет всегда рядом с тобой
✅ **Защита** - атакует врагов и защищает тебя
✅ **Уникальная модель** - красивая 3D модель собаки с текстурой

## Как использовать

### 1. Установка

**Требования:**
- Java 21+
- Minecraft 1.21.1
- Fabric Loader 0.15.11+

**Установка мода:**
1. Скачай файл `grechka-mod-1.0.0.jar`
2. Положи его в папку `.minecraft/mods/`
3. Запусти Minecraft с профилем Fabric 1.21.1
4. Готово! 🎉

### 2. Использование в игре

**Как приручить собаку:**
1. Возьми **кость** в руку
2. Кликни правой кнопкой мыши на собаку
3. Собака приручена! ❤️

**Что собака делает:**
- Следует за тобой везде
- Атакует враждебных мобов
- Не может быть убита (бессмертна)
- Защищает тебя в бою

## Сборка из исходного кода

```bash
# Клонируй репозиторий
git clone https://github.com/yourusername/grechka-mod.git
cd grechka-mod

# Соберешь мод
./gradlew build

# Готовый JAR файл будет в build/libs/
```
